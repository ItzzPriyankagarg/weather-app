# 🌦️ Weather App

A responsive Weather App built using **HTML, CSS, and JavaScript** that provides real-time weather information for any city using a weather API.

---

## ✨ Features

- 🔍 Search weather by city name
- 🌡️ Displays current temperature
- ☁️ Shows weather conditions (Clear, Rain, Clouds, etc.)
- 💧 Displays humidity
- 🌬️ Shows wind speed
- 📱 Responsive design for desktop and mobile
- ⚡ Fast and user-friendly interface

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Weather API

## 🚀 Getting Started

1. Clone the repository:

```bash
git clone https://github.com/ItzzPriyankagarg/weather-app.git
```

2. Open the project folder.

3. If you're using an API key, add it to `script.js` or your configuration file.

4. Open `index.html` in your browser.
